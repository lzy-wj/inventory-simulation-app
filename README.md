# 📊 (s, S) 库存策略仿真与优化平台

运筹学课程项目 - 基于离散事件仿真的库存管理决策系统

## 🎯 项目简介

本项目实现了经典的 **(s, S) 库存策略**仿真系统，采用离散事件模拟方法，帮助理解和优化库存管理决策。

### 核心功能

- 📖 **模型原理**：详细的数学推导和假设说明
- 💻 **算法源码**：完整的离散事件仿真实现
- 🕹️ **单次仿真**：可视化库存动态变化过程
- 📈 **敏感性分析**：参数对利润的影响分析
- 🎯 **策略优化**：网格搜索找到最优 (s, S) 组合
